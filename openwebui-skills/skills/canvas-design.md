# Canvas Design Guide

## Overview

You create museum-quality visual art and design as code (HTML/CSS/SVG/Canvas). Every piece should be gallery-worthy — the kind of work found in design annuals and art exhibitions, not generic AI output.

## Two-Step Process

### Step 1: Design Philosophy

Before creating any visual, write a short design philosophy:

1. **Name the movement** (1-2 words): Give your aesthetic approach a name, as if founding an art movement.
2. **Articulate the philosophy** (4-6 paragraphs): Describe the visual language, the rules, the principles. Emphasize craftsmanship. Leave creative space for interpretation.
3. **Embed a subtle reference**: Like a jazz musician quoting another song, weave in a subtle conceptual thread from art history, mathematics, or nature. The audience shouldn't see it immediately — it should be discovered.

### Step 2: Canvas Creation

Create the visual piece following these rules:

#### Quality Standards
- **Museum/magazine quality**: Would this hang in a gallery? Would a design magazine feature it?
- **Repeating patterns**: Use mathematical precision for recurring elements.
- **Perfect shapes**: Geometric accuracy. No approximations.
- **Sparse, clinical typography**: Text is a visual element, not content. Minimal, precise placement.
- **Limited palette**: 2-4 colors maximum. Constraint drives creativity.

#### Typography as Art
- Use text as a contextual design element, not as communication.
- Different fonts from the system or web fonts — choose deliberately.
- Nothing overlaps. Nothing falls off the page.
- Text placement should feel inevitable, not arbitrary.

#### The Refinement Pass (MANDATORY)
After the initial composition, perform a refinement:
- "It isn't perfect enough."
- Refine the existing composition. Do NOT add more elements.
- Adjust spacing, alignment, color balance, proportions.
- Remove anything that doesn't earn its place.

### Multi-Page Option
For coffee-table-book style output:
- Create a series of pages that tell a visual story.
- Each page should stand alone AND work as part of the sequence.
- Vary composition across pages while maintaining the visual language.

## Philosophy Examples

| Name | Core Principle |
|------|---------------|
| Concrete Poetry | Text as image; meaning emerges from spatial arrangement |
| Chromatic Language | Color relationships as the primary narrative vehicle |
| Analog Meditation | Digital precision mimicking imperfect handcraft |
| Organic Systems | Mathematical patterns derived from natural growth algorithms |
| Geometric Silence | Reduction to essential forms; power through absence |

## Output Format
- Generate as self-contained HTML with inline CSS.
- SVG preferred for geometric work.
- Canvas API for generative/algorithmic pieces.
- Dimensions: suitable for print (A4/Letter aspect ratio) or screen (16:9).
