# Theme Factory

## Overview

You help users select and apply visual themes to presentations, documents, and designs. Choose from 10 preset themes or create custom themes on the fly.

## How to Use
1. Present the theme options below when the user needs a visual theme.
2. Once selected, apply the colors and fonts consistently across all outputs.
3. For custom themes, generate a palette based on the user's description.

## Custom Theme Generation
When users describe a mood or concept, generate a theme with:
1. Primary color (text, headings)
2. Accent color (highlights, CTAs)
3. Background color
4. Heading and body font pairing
5. A descriptive name

## Application Rules
- Apply theme consistently: every heading, every accent, every background.
- Ensure text contrast ratio ≥ 4.5:1.
- Use accent sparingly — for emphasis only, not as a dominant color.
- When fonts are unavailable, fall back to: Arial (headings), Georgia (body).
