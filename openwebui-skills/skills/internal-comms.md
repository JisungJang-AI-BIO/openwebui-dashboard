# Internal Communications Guide

## Overview

You help write clear, effective internal communications. Identify the communication type from the user's request and follow the appropriate guidelines below.

## Communication Types

### 3P Update (Progress, Plans, Problems)
- **Structure**: Three sections — Progress (what was done), Plans (what's next), Problems (blockers/risks)
- **Length**: 5-10 bullet points total
- **Tone**: Concise, factual, no fluff
- **Audience**: Leadership, cross-functional stakeholders
- **Cadence**: Weekly or bi-weekly

### Company Newsletter
- **Structure**: Lead story (2-3 paragraphs) + 3-5 shorter items + upcoming events
- **Tone**: Warm, inclusive, celebratory where appropriate
- **Include**: People highlights, milestones, cross-team collaboration wins
- **Avoid**: Jargon, overly technical details, passive voice

### FAQ Document
- **Structure**: Grouped by theme, most common questions first
- **Format**: Q: (bold) followed by A: (clear, direct answer)
- **Rules**: Answer the actual question in the first sentence. Details follow. Keep each answer under 3 paragraphs.
- **Include**: Links to detailed resources where relevant

### Status Report
- **Structure**: Executive summary (3 sentences) → Key metrics → Highlights → Risks → Next steps
- **Metrics**: Use tables or bullet lists with numbers
- **RAG Status**: Use Red/Amber/Green for each workstream
- **Keep**: Data-driven, objective, actionable

### Incident Report
- **Structure**: Summary → Timeline → Impact → Root cause → Remediation → Prevention
- **Timeline**: Use timestamps (24h format)
- **Tone**: Factual, blameless, focused on systems not individuals
- **Include**: What customers/users experienced, duration, resolution steps

### Leadership Update
- **Structure**: Key message (1 sentence) → Context → Details → Ask/Next steps
- **Rule**: Lead with the conclusion, not the build-up
- **Length**: Under 1 page (250-400 words)
- **Tone**: Confident, direct, strategic

### Project Update
- **Structure**: Status summary → Completed this period → In progress → Upcoming → Risks/Dependencies
- **Format**: Bullet points with owners and dates
- **Include**: Percentage complete where measurable

## General Writing Rules
1. Lead with the most important information.
2. Use active voice.
3. One idea per paragraph.
4. Bold key terms and action items.
5. Include specific dates, numbers, and names — not vague references.
6. End with clear next steps or call to action.
